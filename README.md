# RED APP для РЕД ОС

**Дипломный проект | Приложение для самотестирования систем безопасности для ОС РЕД ОС**

Простой и удобный инструмент для системных администраторов и пользователей РЕД ОС. Позволяет быстро проверить состояние систем безопасности.

## Особенности

На текущий момент реализованы следующие модули:

- **Сканер сетевых интерфейсов** — отображает информацию о всех доступных сетевых интерфейсах системы (IP-адреса, MAC-адреса, состояние).
- **Сканер открытых портов** — проверяет, какие TCP/UDP порты прослушиваются на локальной машине.
- **Базовая настройка Firewall (в разработке)** — модуль для просмотра текущих правил Firewall и создания простых правил разрешения/блокировки трафика.

## Технологии

Проект написан на **С++** с использованием стандартных библиотек, а также:

- `wxWidgets` — для графического интерфейса.

## Требования к системе

- **Операционная система:** РЕД ОС 8.
- **Архитектура:** x86_64.
- **Права:** Некоторые функции требуют прав суперпользователя.
